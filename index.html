📌 Como configurar a Supabase (passo a passo)
  1. Vai a supabase.com e cria uma conta gratuita
  2. Cria um New Project (dá-lhe um nome, ex: fichas-custo)
  3. Aguarda ~2 minutos até o projecto estar pronto
  4. Vai a SQL Editor e corre o ficheiro supabase_setup.sql que te foi fornecido
  5. Vai a Settings → API e copia:
      • Project URL (ex: https://abc123.supabase.co)
      • Publishable key (começa com sb_publishable_) ou anon key (começa com eyJ)
  6. Cola as duas chaves aqui em cima e clica Guardar e ligar
  7. Partilha este ficheiro HTML com os colegas — todos ligam à mesma Supabase
